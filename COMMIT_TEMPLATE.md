# Commit Message Template

The following template should be used for all commit messages in this repository. The body/description must be an unordered list of what was done. GitHub Copilot agent will read this file before committing.

```
<Title: concise summary of what was done>

- <First change or addition>
- <Second change or addition>
- <Additional changes as needed>
```

## Example
```
Refactor MedicationsStock API

- Renamed register to new for constructor clarity
- Clarified add_stock for stock registration
- Updated docs and tests accordingly
```
